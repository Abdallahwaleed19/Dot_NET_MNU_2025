# C# Project - Dot_NET_MNU_2025

## 📌 Description
This repository contains C# projects and exercises developed as part of the **.NET course at Menoufia National University**.  
It covers different C# concepts including OOP, Collections, LINQ, Exception Handling, and more.

---

## 🚀 Features
- Object-Oriented Programming (OOP)
- Data Structures & Collections
- LINQ Queries
- Exception Handling
- File Handling
- Windows Forms (if applicable)
- .NET Framework / .NET Core

---

## 🛠️ Technologies Used
- **Language:** C#
- **Framework:** .NET (Core/Framework)
- **IDE:** Visual Studio
- **Version Control:** Git & GitHub
